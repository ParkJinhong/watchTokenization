A. 토큰화 시장

시계를 분할 토큰으로 — 발행 · 투자 · 수익 배분

고가의 명품 시계를 여러 조각(토큰)으로 나눠 누구나 소액으로 투자하고, 대여·거래로 수익을 만든다.

1. 핵심 흐름

1시계 공급보유분 / 자금 모집 구입·중개
→
2감정·등급컨디션 → S·A·B·C
→
3토큰 발행1점 = 분할 지분 토큰
→
4투자자 매수소액 분할 투자
→
5거래·수익2차 거래 / 대여 배분

2. 단계별 상세

① 시계 공급 — 어떻게 물량을 확보하나오프체인
  • 회사 보유 재고: 시계거래소가 이미 보유한 시계를 토큰화
  • 자금 모집 후 구입: 투자자 자금을 먼저 모아 시장에서 시계 매입
  • 토큰화 희망자 중개: 개인이 맡긴 시계를 토큰화해 주고 수수료 수취
② 감정·등급 — 같은 모델도 컨디션이 다르다오프체인

전문 감정으로 컨디션을 4등급으로 표준화하여 토큰 가격의 기준을 만든다.

등급기준(예시)비고
S미사용/풀세트, 흠집 없음프리미엄 가격
A경미한 사용감, 정품 보증서 보유표준
B사용감 뚜렷, 일부 부품 교체할인
C외관 손상/수리 이력큰 할인
③ 토큰 발행 — 1점을 1,000조각으로온체인
  • 시계 1점 = 분할 지분 토큰(ERC-20)으로 발행. 총발행량이 곧 100% 지분
  • 등급·식별자(감정서 번호)를 토큰에 기록 → 어떤 실물에 연결되는지 추적 가능
  • 증권형으로 분류되면 전송 제한이 가능한 표준(ERC-3643)으로 확장
④ 투자자 매수 — 왜 토큰을 사나
  • 분할 접근성: 통째로 못 사는 고가 시계를 소액으로 일부 보유
  • 투자(시세 차익): 시계 가치 상승 시 토큰 가격도 상승 기대
  • 대여 수익: 보유만으로 대여 수익을 정기 배분받음
⑤ 거래·수익 — 두 갈래온체인
  • 2차 거래: DeFi AMM 또는 P2P 거래소에서 토큰을 사고팖 (증권형이면 인가 거래소만)
  • 대여 수익 배분: 시계를 대여하고 발생한 수익을 토큰 보유 비율대로 분배(claim 방식)

3. 수익 배분 구조

대여 수익은 보유자가 많을수록 일일이 보내기 비효율적이므로, 각자 직접 청구(pull-claim)하는 방식이 적합하다.

구분내용
수익 원천시계 대여료 (오프체인 발생 → 정산 통화로 컨트랙트에 입금)
배분 기준특정 시점 토큰 보유량 스냅샷 비례
수령 방식보유자가 원할 때 자신의 몫을 청구(claim)

4. 예시 시나리오

롤렉스 서브마리너 (감정가 1억 원, S등급)

발행1,000 토큰 발행 → 1토큰 = 10만 원
투자100만 원 투자 시 10토큰 = 시계 지분 1%
대여 수익연 대여수익 600만 원 → 토큰당 6,000원 배분 (연 6% 수익률)
시세 차익시계 감정가 1.2억으로 상승 시 → 1토큰 12만 원 (+20%)

5. 회사 수익 모델

회사 수익 거래 수수료(매매 시) + 대여 수수료(대여 수익의 일정 %)

6. 관련 컴포넌트

컴포넌트역할위치
WatchToken시계 분할 지분 토큰(등급·식별자 기록)온체인
RevenueDistributor대여 수익 스냅샷·청구 분배온체인
거래소(AMM/P2P)토큰 2차 거래온체인
감정·등급 엔진컨디션 평가, 가격 산정오프체인

7. 리스크 포인트

증권성 — "대여 수익 배분 + 투자 유인"은 한국 자본시장법상 투자계약증권으로 볼 가능성이 크다. 이 경우 발행·유통에 인가가 필요하고, 퍼미션리스 거래소 자유 거래가 제한된다. (상세: 기획·리스크 문서 3장)
